# Attendance Frontend

React + Vite frontend for the Student Attendance Management System.

## Features

- Role-based dashboard flows
- Authentication screens
- Admin management screens
- Analyst batch management
- Counsellor student management
- Student attendance views
- Camera and face-verification related UI

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Dev Server

By default, the frontend runs at `http://localhost:5173/`.

Make sure the Django backend is running so the frontend can call the API successfully.
